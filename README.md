
# Gestão de fornecedores

## Executar back-end localmente

Vá para o diretório do **server**:

```bash
  cd server
```

Instalar dependências:

```bash
  yarn install
```

Inicie o servidor em modo de produção:

```bash
  yarn start:prod
```

## Executar front-end localmente

Vá para o diretório do **web**:

```bash
  cd web
```

Instalar dependências:

```bash
  yarn install
```

Inicie o servidor em modo de produção:

```bash
  yarn start
```
